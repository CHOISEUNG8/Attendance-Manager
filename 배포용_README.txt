프로그램 실행 방법:
------------------
1. "WB_Attendance_Manager_v4_1.exe" 파일을 더블클릭하여 실행하세요.

2. 첫 실행 시 10-30초 정도 걸릴 수 있습니다. (정상적인 현상)

3. 프로그램이 자동으로 데이터베이스 파일(leave_attendance.db)을 생성합니다.


주요 기능:
----------
- 재직인원 관리 (직원 추가, 삭제, 퇴사 처리)
- 출퇴근 관리대장 (일별 출퇴근 기록, 요약 통계)
- 연월차 관리대장 (연차 발생, 사용, 잔여수 관리)
- 엑셀 업로드/다운로드 지원


데이터 백업:
-----------
중요: 데이터를 보존하려면 다음 파일을 정기적으로 백업하세요:
- leave_attendance.db (데이터베이스 파일)

이 파일은 프로그램과 같은 폴더에 생성됩니다.


문제 해결:
----------
1. 프로그램이 실행되지 않는 경우:
   - 관리자 권한으로 실행해보세요
   - Windows Defender에서 예외 처리하세요

2. 오류가 발생하는 경우:
   - 프로그램을 다시 시작해보세요
   - 데이터베이스 파일(leave_attendance.db)을 백업한 후 삭제하고 다시 실행해보세요

3. 엑셀 파일이 열리지 않는 경우:
   - Microsoft Excel이 설치되어 있는지 확인하세요
   - 파일이 다른 프로그램에서 열려있지 않은지 확인하세요


문의사항:
--------
프로그램 사용 중 문제가 발생하면 개발자에게 문의하세요.
